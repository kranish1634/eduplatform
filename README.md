# EduPlatform

Deployed on Render:-  https://eduplatform-frontend-jdmz.onrender.com

EduPlatform is a comprehensive full-stack web application designed to facilitate seamless online learning and course management. 
Built with a modern technology stack, the frontend utilizes React and Vite to deliver a fast, responsive, and highly interactive user experience. 
The backend is powered by Node.js and Express.js, providing a robust, scalable RESTful API to handle operations efficiently. 
Data is managed using MongoDB and Mongoose, with cleanly defined schemas for users, courses, admins, and platforms interactions. 
Security is a core focus, leveraging JSON Web Tokens (JWT) and bcryptjs to ensure secure authentication and data protection.
A rigid Role-Based Access Control (RBAC) architecture strictly governs permissions, routing users dynamically based on their admin, instructor, or student roles.
The platform includes built-in dynamic PDF generation using tools like `jspdf` and `html2canvas` for seamless certificate exports.
Backend workflows are highly optimized with custom utility scripts that easily handle tasks like database seeding and password resets.
The entire codebase is structured with clear separation of concerns, maintaining cleanly isolated controllers, middleware, and utility functions.
Designed with long-term maintainability in mind, EduPlatform acts as a reliable, fully-featured foundation for modern educational technology projects.
